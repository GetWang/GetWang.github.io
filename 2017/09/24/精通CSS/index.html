精通CSS

发表于 2017-09-24 | 分类于 CSS 总结

选择器

  • 链接伪类： :link和:visited，只能应用于锚元素
  • 动态伪类： :hover、:active和:focus，理论上可以应用于任何元素
  • IE6只注意应用于锚链接的:active和:hover；完全忽略:focus
  • IE7在任何元素上都支持:hover，但是忽略:active和:focus
  • 通配符选择器“*”的优先级最低

注释

为了让注释更有意义，可用关键字来区分重要的注释，用编程术语来说，这些关键字称为gotcha，它们在以后的开发阶段非常有帮助。参考CSSDoc项目

外边距叠加

当两个或更多垂直外边距相遇时，它们将形成一个外边距，这个外边距的高度等于发生叠加的外边距的高度的最大者。外边距可以与本身发生叠加。

浮动

  • 要清除某父容器内的浮动元素，可以让父元素也浮动起来，从而达到清除浮动的效果
  • 应用值为hidden或auto的overflow属性有一个有用的副作用，这会自动清除父元素内包含的任何浮动元素

在导航条中突出显示当前页面

不需要动态地在菜单（导航）中添加类的解决方案：在每个页面的主体元素中添加一个ID或类名，从而指出用户当前在哪个页面或部分中；然后，在导航列表中的每个项中添加一个相应的ID或类名；就可以使用主体（body）的ID/类和列表ID/类的唯一组合在站点导航中突出显示当前页面。eg：

表格

在一个表格中只能使用一个thead和tfoot，但可使用多个tbody；且如果选择使用thead或tfoot，那么至少必须使用一个tbody元素

标签<label>

有时我们不希望为每个表单控件元素都显示标签，这时我们可以让标签出现的源码中但不显示在页面中。使用文本缩进，标签宽度置为0

修复bug

  • 有时，仅仅添加边框就能修复问题（往往说明存在外边距叠加问题）
  • 尝试将bug的效果扩大化
  • 将position设为relative
  • 将displa设置为inline（在浮动元素上）
  • 设置宽度等尺寸

字体格式

  • IE只支持.eot(Embedded OpenType)字体格式
  • .ttf(TrueType)转为.eot
  • .otf(OpenType)等其它字体格式转为.ttf，再转为.eot

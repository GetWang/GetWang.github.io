设置 overflow 属性为 hidden

发表于 2017-09-24 | 分类于 一些 CSS 开发技巧

用途

当父元素中一部分子元素浮动，另一部分子元素不浮动，并且想清除浮动，这时可在“另一部分”子元素上设置 overflow: hidden 来清除浮动。此外，利用它可使浮动子元素与不浮动子元素并排显示（宽度允许的话），而 clear: both 却做不到。

原理

触发了 BFC 模式（Block Formatting Context）块级格式化文本。BFC 中的内容与外界的元素是隔离的。

精通 CSS

发表于 2017-09-24 | 分类于 CSS 总结

选择器

  • 链接伪类：:link 和 :visited，只能应用于锚元素
  • 动态伪类： :hover、:active 和 :focus，理论上可以应用于任何元素
  • IE6 只注意应用于锚链接的 :active 和 :hover；完全忽略 :focus
  • IE7 在任何元素上都支持 :hover，但是忽略 :active 和 :focus
  • 通配符选择器 \* 的优先级最低

标签使用注意事项

发表于 2017-09-24 | 分类于 HTML

  • 有几个特殊的块级元素智能包含内联元素，不能包含块级元素，分别是 h1-h6、p、dt

  • 块级元素可以包含内联元素或某些块级元素，但内联元素绝对不能包含块级元素

  • <img> 元素有 margin 值，故初始化时，会将其 display 为 block
